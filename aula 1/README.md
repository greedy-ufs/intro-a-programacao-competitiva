# Aula 1

## Tópicos:

- C++
- STL
- Complexidade

## Resumo:

Nessa aula, vimos:
- A estrutura de um programa em C++
- Como podemos usar os comando de C em C++
- Compilação e execução de um programa em C++

Vimos também as seguintes estruturas de dados da STL:
- vector
- queue
- stack
- set
- map

E por último, aprendemos como:
- Analisar a complexidade de algoritmos e
- Estimar o tempo de execução da nossa solução

Com isso, sabemos, antes mesmo de codar, se a nossa solução passaria ou não dentro do tempo determinado pela questão.
